# Understanding Playwright

## Overview
This repository contains my personal Playwright automation testing practice projects, created to strengthen my understanding of Playwright syntax, test structure, and modern Quality Engineering principles.

The project focuses on building effective and maintainable end-to-end automated tests using Playwright with TypeScript.

---

## Applications Under Test

This repository includes tests for:

- A web-based application developed for our Quality Engineering coursework
- Public websites used for automation practice, including:
  - Nike
  - Adidas
  - Flash Group South Africa

---

## Learning Goals

The purpose of this repository is to:

- Improve practical Playwright automation skills
- Write clean and maintainable test scripts
- Understand real-world UI automation workflows
- Practice scalable automation framework design
- Explore advanced Playwright features and best practices
